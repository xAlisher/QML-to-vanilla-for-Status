## Key Differences from v2
- Neutrals are blue-tinted (navy undertones like `#1B273D`) vs v2's pure grays (`#2C2C2C`)
- Success is teal (`#23ADA0`) vs v2's green (`#4EBC60`)
- Primary blue is the same (`#4360DF`)
- Danger is warmer red (`#E95460`) vs v2's pink-red (`#FF2D55`)
- Much deeper dark mode (navy `#09101C` vs gray `#2C2C2C`)
